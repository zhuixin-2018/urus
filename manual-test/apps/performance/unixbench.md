#命令说明
```
       -i # 用来指定测试内容.
       0=write/rewrite
       1=read/re-read
       2=random-read/write
       3=Read-backwards
       4=Re-write-record
       5=stride-read
       6=fwrite/re-fwrite
       7=fread/Re-fread,
       8=random mix
       9=pwrite/Re-pwrite
       10=pread/Re-pread
       11=pwritev/Re-pwritev,
       12=preadv/Re-preadv
```
#选项
---
　　　　　　　-R 产生execl格式的输出日志。

              -b 将产生二进制的execl的日志文件名。

              -s 测试的文件大小。

              -q 指定最大文件块大小（这里的 -q 64k 包括了4K,8K,16K,32K,64K）

              -r 指测试的文件块大小（与-q有别，-r 64k只进行64k的测试）

              -a 在希望的文件系统上测试，不过只有
a的话会进行全面测试，要花费很长时间，最好用-i指定测试范围。

             -g 指定最大测试文件大小。

             -n 指定最小测试文件大小。

             -f 指定测试文件的名字，完成后会自动删除（这个文件必须指定在你要测试的那个硬盘中）

             -C 显示每个节点的吞吐量。

             -c 测试包括文件的关闭时间
